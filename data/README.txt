DOWNLOAD REQUIRED: Apple 10-K Filing

This directory should contain Apple's 10-K filing named: apple_10k_2023.htm

To download:
1. Visit: https://www.sec.gov/cgi-bin/browse-edgar?action=getcompany&CIK=0000320193
2. Find the most recent 10-K filing (NOT 10-K/A)
3. Click "Documents" and download the HTML version
4. Save it as: apple_10k_2023.htm in this directory

The file is not included in this ZIP due to its large size (~5-10 MB).
